# qiankun-vue3-ts-vue-cli5 

该模板使用 Vue3 + cli5 + Typescript + qiankun


**包管理器推荐使用 pnpm**

> 请先`npm install pnpm -g`；

## _安装运行_

```bash
npm install pnpm -g
pnpm install-all
pnpm run dev
# 打开 http://localhost:8990/
```
